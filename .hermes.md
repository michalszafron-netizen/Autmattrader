## TradingView MCP — kluczowe funkcje

### Czyszczenie rysunków z wykresu (JEDYNA działająca metoda)
```javascript
// Użyj ui_evaluate z dokładnie tym kodem (model() jest WYMAGANE):
_exposed_chartWidgetCollection.activeChartWidget._value.model().removeAllDrawingTools()
```
Wywołanie: `ui_evaluate` z powyższym kodem. NIE używaj innych metod.
UWAGA: `.value.removeAllDrawingTools()` — BEZ model() — NIE DZIAŁA. Zawsze `.value.model().removeAllDrawingTools()`

### Zmiana symbolu / timeframe
```
chart_set_symbol("BINANCE:BTCUSDT")
chart_set_timeframe("240")   // 4H
```

### Rysowanie poziomów
- TP: `draw_shape(horizontal_line)` kolor #22ab94, linewidth 2
- SL: `draw_shape(horizontal_line)` kolor #f23645, linewidth 2
- Strefa entry: `draw_shape(rectangle)` fill #22ab9433 (LONG) lub #f2364533 (SHORT)

### Kolejność operacji (ZAWSZE)
1. `chart_set_symbol()` — przełącz
2. `chart_get_state()` — poczekaj na gotowość
3. `ui_evaluate(removeAllDrawingTools)` — wyczyść
4. Rysuj poziomy
